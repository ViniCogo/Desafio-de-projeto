O que são tags HTML(Voltar)

Tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”.

Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing).

Tags de texto(Voltar)

  • Blockquote - É utilizado para fazer uma citação, distanciando o texto da borda e separando ele do restante do texto

  • H's - São hider's, voltados para títulos, subtítulos e assim por diante. "<h1>" é o maior e seus sucessores são cada vez menores.

  • Mark - Tag utilizada para marcar um texto cobrindoo com uma cor diferente.

  • P - A tag "<p>" é utilizado para adição de parágrafos.

    A cada quebra de linha ou ponto final, deve ser adicionado um novo P.

  • Sup - A tag "<sup>" utilizada para adicionar um texto superior a uma palavra, como por exemplo uma elevação ao quadrado 2².

  • U - A tag "<u>" é a abreviação de Underline, que é a tag usada para sublinhar um texto.

  • I - A tag "<i>" é usada para deixar um texto no formato italico.

  • Strong - A tag "<strong>" é usada para deixar um texto destacado em negrito

  • Hr - A tag "<hr>" é usada para criar uma linha que separa a tela como exemplo adicionarei uma abaixo.

Tags de lista(Voltar)

A lista pode ou não ser ordenada.

Se a lista for ordenada a tag será "<ol>".

É necessário adicionar dentro da tag ol, a tag "<li>" (List iten), se a lista for maior do que 1, as tags devem ser adicionadas uma sobre a outra.Esta lista é enumerada.

A tag para lista não ordenada é <ul>, diferente da lista ordenada, ela adiciona marcadores no lugar de números.
